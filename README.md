# :books: LiterAlura

Este repositório contém um projeto desenvolvido utilizando Java, Spring e Noçẽs de Banco de Dados. 
Este projeto é o desafio da Formação em Back-End da Oracle ONE: TECH FOUNDATION em parceria com Alura.
Foi utilizado a API Gutendex para resgatar as informações necessarias dos livros e, apesar do desafio pedir para usar PostgreSQL, neste caso usei H2 Database Engine.

# :computer: O Que Faz.
É possível pesquisar um livro por autor, idioma ou titulo,bem como listar os 10 livros mais baixados, listar os livros e os autores que foram salvos no banco.
